# Product Showcase Grid

A Flutter UI project that showcases a modern product browsing experience using a GridView layout. This project focuses on reusable widgets, navigation, and basic state management with `setState`.

## Features

-  Product Grid Layout
-  Product Detail Screen
-  Favorite Toggle using `setState`
-  Reusable Widgets
-  Bottom Navigation Bar
-  Clean and Responsive UI

## Built With

- Flutter
- Dart
- Material Design

## What I Learned

- Building responsive GridView layouts
- Creating reusable widgets
- Navigation between screens
- Managing UI state using `setState`
- Organizing Flutter project structure
